News & Tools

🆕 TEHDAS2 Guideline D7.1 — Finalisation Underway: How to Use Data in a Secure Processing Environment (SPE)
TEHDAS2's definitive SPE guideline (D7.1) is moving towards final publication in H1 2026. The document defines common security requirements, functional and technical services that all EHDS-compliant SPEs must provide, and stipulates that only anonymised results can be exported — forming the de facto regulatory baseline for any Kubernetes/cloud-native SPE deployment.
📅 Final versions expected H1 2026 · tehdas.eu · Draft D7.1 PDF

The European Commission published the source code of the HealthData@EU Central Platform on code.europa.eu, providing reusable open-source components for Member States to connect national health data catalogues, access points, and HDABs to the EU-level EHDS secondary-use infrastructure.
